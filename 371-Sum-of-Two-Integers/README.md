# 371. Sum of Two Integers

**Easy**

[Original Page](https://leetcode.com/problems/sum-of-two-integers/)

Calculate the sum of two integers a and b, but you are __not allowed__ to use the operator `+` and `-`.

##### Example 1:
```
Input: a = 1, b = 2
Output: 3
```

##### Example 2: 
```
Input: a = -2, b = 3
Output: 1
```
